# 금투분석사 개인 오답회수기 v0.9

개인용 모바일웹/PWA 학습 도구입니다. 기본 내장 A~H 타입 문제은행, 랜덤 풀이, 오답 복습, 실제 과목 순서 모의고사 중심으로 구성했습니다.

## v0.9 변경사항

- 답안해설에서 `4. 단계별 체크` 섹션을 제거했습니다.
- 답안해설 번호를 `4. 보기별 판별`, `5. 오답 포인트`, `6. 시험장 시간단축 팁`, `7. 암기팁`으로 정리했습니다.
- 모의고사 답안해설에서 펼친 문항의 본문 아무 곳이나 클릭하면 다시 제목만 보이도록 접히게 했습니다.
- 답안해설 영역의 여백, 줄간격, 선택지 간격을 줄여 모바일 화면 낭비를 줄였습니다.

## 유지 기능

- A~H 문제은행
- 랜덤 5/10/20/50문제 풀이
- A~H 유형별 랜덤 풀이
- 실제 과목 순서 모의고사
- 모의고사 타이머, 상단 120분 타이머, Reset 버튼
- 모의고사 결과의 맞힌 문제/틀린 문제 구분 답안해설
- 문항별 풀이시간 표시
- 힌트 보기/숨기기
- 다크모드/라이트모드
- JSON 백업/복원

## 실행 방법

### 단일 HTML

`fta-personal-study-app-v0.9-standalone.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

### PWA/정적 배포

프로젝트 폴더에서 아래 명령을 실행하세요.

```bash
python3 -m http.server 8000
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8000
```

## 데이터 저장

모든 풀이 기록, 오답노트, 설정은 브라우저 로컬 저장소에 저장됩니다. 캐시 삭제나 기기 변경 시 데이터가 사라질 수 있으니 설정 화면의 JSON 백업을 사용하세요.
